Desenvolvimento Web 2021

    Este trabalho foi desenvolvido como projeto final, para a disciplina Desenvolvimento Web 1, no curso de Sistemas de informação da Universidade Federal do Mato Grosso UFMT Campus Cuiabá!

    Neste projeto, onde contemplamos a parte FrontEnd: HTML, CSS e JavaScript, assim como o BackEnd: PHP e SQL, foi desenvolvido a página principal, a página Usuários, onde estão listados todos os usuários criados no banco de dados, assim como o botão para criar usuário, que nós leva a uma página não listada no menu, contendo o formulario de criação.
